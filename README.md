# Origins

A concise React storefront built with Vite. Provides Home, Products, Category, Product detail, Cart and Checkout. Optional Clerk authentication.

## Features
- Responsive product listing, category filters, single-product pages, cart and checkout.
- Newsletter (react-hook-form), accessible FAQ accordion, scroll-to-top behavior.

## Tech Stack
React, Vite, React Router, Clerk, react-hook-form, CSS Modules.

## Quick start
1. Install: `npm install`
2. Run dev server: `npm run dev`
3. Build: `npm run build`

## Environment
- `VITE_CLERK_KEY` — set to enable Clerk authentication; when absent the app runs without auth.

## Layout
- `src/` — application source; `src/components/` holds UI components; `src/assets/` contains images/icons.
